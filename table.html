Unix Commands

Terms Definition
cd The command used to change directories.
pwd The command used to see what current directory you're in.
mkdir The command used to make a new directory.
nano The command used to create a new text file.
cp -r The command used to copy recursively.
rm -fr The command used to forcefully remove a file or directory.
sudo The command that gives you temporary super-user access.
